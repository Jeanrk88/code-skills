# Jeanrk88 Code Skills

Pack público de skills para Codex focado em criar softwares com mais qualidade, melhor UX, melhor engenharia, QA real e segurança prática.

## O que este repositório entrega

Este repositório reúne skills reutilizáveis para acelerar fluxos de desenvolvimento com padrão mais alto de qualidade em quatro frentes:

- **frontend-design** — criação e validação de interfaces frontend memoráveis, production-grade e com direção estética forte.
- **software-engineer** — auditoria de arquitetura, segurança, performance, resiliência e custo.
- **qa** — verificação prática de testes, gaps, infraestrutura, performance, custo e acessibilidade básica.
- **security** — análise e fortalecimento de segurança com stack detection, scanner OWASP, secret scanning, monitoramento e relatório.

## Skills incluídas

### frontend-design

Cria e valida interfaces frontend distintivas, memoráveis e production-grade com alto padrão visual, craft, experiência e implementação.

### software-engineer

Audita código em nível world-class em todas as camadas: arquitetura, segurança, performance, resiliência, custo, infraestrutura e escala.

### qa

Verifica que tudo funciona na prática: testes, gaps de cobertura, infraestrutura, integridade de dados, performance, custo e acessibilidade básica.

### safe-e2e-test

Executa ou orienta testes end-to-end seguros, focando em fluxos críticos, previsibilidade e prevenção de efeitos colaterais destrutivos.

### security

Aplica workflow de segurança com:

- reconhecimento automático de stack
- varredura OWASP Top 10
- pen test simulado
- correção guiada
- proteções HTTP/CSRF/rate limit/sanitização
- firewall
- monitoramento contínuo
- relatório final com score
